
## 1. Run Backend (Node.js)
```
# go to the server folder
cd server

# install dependencies
npm install

# start backend server
node server.js
```
* Backend defined port: `3001`
* On one terminal
---

## 2. Run Frontend (Vite)
```
# install dependencies
npm install

# start Vite dev server
npm run dev
```
* Frontend defined port: `3000`
* On another terminal
---
